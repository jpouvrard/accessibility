Déclaration d'accessibilité

Cette déclaration s'applique au contenu publié sur le domaine :

Elle ne s'applique pas aux autres contenus ou sites web publiés sur ses sous-domaines. Ces sites web et leur contenu auront leur propre déclaration d'accessibilité spécifique.

Ce site web est géré par la Commission européenne au nom de toutes les institutions de l'Union européenne, sous la supervision du Comité interinstitutionnel de communication en ligne (IOCC). Il est conçu pour être utilisé par le plus grand nombre possible de personnes, y compris les personnes handicapées.

Vous devriez pouvoir :

  • zoomer jusqu'à 200 % sans problèmes
  • naviguer sur la majeure partie du site web en n'utilisant que le clavier
  • naviguer sur la majeure partie du site web à l'aide d'un lecteur d'écran moderne et d'un logiciel de reconnaissance vocale (sur votre ordinateur ou téléphone)

Ce site web est conçu pour être conforme à la norme technique pour les sites web et les applications mobiles, EN 301 549 v.3.2.1. Celle-ci suit de près le niveau 'AA' des Directives pour l'accessibilité des contenus Web (WCAG) version 2.1.

État de conformité

Ce site web est partiellement conforme à la norme technique EN 301 549 v.3.2.1 et aux Directives pour l'accessibilité des contenus Web (WCAG) 2.1 Niveau AA. Voir 'Contenu non accessible' pour plus de détails.

Le site web a été testé pour la dernière fois le 25 octobre 2024.

Préparation de cette déclaration

Cette déclaration a été préparée le 16 décembre 2024.

La déclaration est basée sur l'examen d'un échantillon représentatif de pages web par un expert en accessibilité, utilisant une combinaison de tests manuels et automatisés.

Compatibilité avec les navigateurs et les technologies d'assistance

Le site web de est conçu pour être compatible avec les technologies d'assistance les plus utilisées suivantes :

  • les dernières versions des navigateurs Google Chrome, Microsoft Edge, Mozilla Firefox et Apple Safari ;
  • en combinaison avec les dernières versions de NVDA, VoiceOver et TalkBack.

Spécifications techniques

L'accessibilité du site web de repose sur les technologies suivantes pour fonctionner avec la combinaison particulière de navigateur web et de technologies d'assistance ou de plugins installés sur votre ordinateur :

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

Contenu non accessible

Malgré nos meilleurs efforts pour assurer l'accessibilité du site web de , nous sommes conscients de certaines limitations, que nous nous efforçons de corriger. Vous trouverez ci-dessous une description des limitations connues et des solutions potentielles. Veuillez nous contacter si vous rencontrez un problème non répertorié ci-dessous.

Limitations connues pour le site web de :

  • Lecteur vidéo – accessibilité au clavier : Nous sommes conscients de certains problèmes avec l'accessibilité au clavier de notre lecteur vidéo (lors de la sélection de la langue des sous-titres dans le menu, il n'est pas possible d'utiliser les touches fléchées pour se déplacer entre les langues, cela ne peut être fait qu'avec la touche TAB).
  • Sous-titres vidéo : Certaines de nos vidéos n'ont pas de sous-titres. Nous travaillons continuellement à en ajouter à davantage de vidéos.
  • Description audio : Les vidéos sur ce site web n'ont pas de description audio.
  • Titre manquant dans un cadre pour la vidéo : Lorsque les vidéos sont intégrées dans un cadre, celui-ci n'a pas toujours un titre. Nous prévoyons de corriger ce problème d'ici fin mars 2025.
  • Hiérarchie des titres : Certaines pages ne présentent pas le contenu dans une organisation hiérarchique des titres. Nous prévoyons de corriger cela d'ici fin mars 2025.
  • Les liens externes s'ouvrent sans avertissement : Certains liens ouvrent une nouvelle fenêtre (ou un nouvel onglet) sans avertissement. Nous prévoyons de corriger cela d'ici fin mars 2025.
  • Liens : Lorsqu'un lien de téléchargement renvoie vers une autre page web, il ouvre une nouvelle fenêtre ou un nouvel onglet. Nous prévoyons de corriger cela d'ici fin mars 2025.
  • Animations : Certaines pages contiennent des animations, comme des graphiques animés, qui ne peuvent pas être mises en pause, arrêtées ou masquées. Nous prévoyons de corriger cela au cours de l'année 2025.
